Scripts and files to build NetBSD/hpcarm version "Teokure Live Image",
pre-installed NetBSD Live Image for Sharp W-ZERO3 smart phones
with typical Japanese environments and useful standard pkgsrc
applications, including browser (dillo) and Twitter client (mikutter) etc.

See http://www.ceres.dti.ne.jp/tsutsui/netbsd/liveimage/wzero3.html
for details.
